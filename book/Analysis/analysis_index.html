组合分析

该页面为组合分析页面，由过滤条件、模糊搜索栏和组合分析栏 3 部分组成。 过滤条件

  • 查询维度：省份、地市、浏览器类型、浏览器、排序方式、最大访问量和平均加载时间；

  • 省份和地市联动；浏览器类型和浏览器联动；排序方式选取值为：最大加载时间和最大访问量;

  • 组合查询维度界定了页面所有图表和子查询的维度；

  • 下拉选择交互样式。

组合分析栏

  • 组合分析栏由访问量 & 访问页面、子查询、URL TOP10 、IP TOP10 、用户标识 TOP10 、地市 TOP10、浏览器 TOP10 七部分组成。

  • 访问量 & 页面加载时间展示的是当前时间段用户访问的访问量和页面加载时间，柱图代表访问量、折线代表页面加载时间。

  • URL TOP10 展示的是当前时间段用户访问的 URL TOP10，由 URL、访问量、占比、加载时间 4 个维度组成。

  • IP TOP10 展示的是当前时间段用户访问的 IP TOP10，由 IP、访问量、占比、加载时间 4 个维度组成。

  • 用户标识 TOP10 展示的是当前时间段用户访问的用户标识 TOP10，由 用户标识、访问量、占比、加载时间 4 个维度组成。

  • 城市 TOP10 展示的是当前时间段用户访问城市的 TOP10，左侧展示的是不同颜色代表不同的城市，并展示出访问量、占比和加载时间。把光标放到右侧某一城市上会显示出当前城市的访问量。

  • 浏览器 TOP10 展示的是当前时间段用户访问所用的浏览器 TOP10，左侧展示的是不同颜色代表不同的浏览器，并展示出访问量、占比和加载时间。把光标放到右侧某一浏览器上会显示出当前浏览器的访问量。

  • 子查询展示的是当选择某一条 URL 时，会展示出当前 URL 的 IP、用户标识 TOP10、城市 TOP10、浏览器 TOP10。或选择一条 URL 选择当前 URL 下的某个 IP，会展示出当前 IP 下的用户标识 TOP10、城市 TOP10、浏览器 TOP10。

过滤条件

  • 当省份选择多个时，地市展示 TOP10，当省份选择为一个时，展示该省的所有地市；

  • 子查询维度：URL、IP、用户标识 属于子查询维度，例如：用户可以选择 URL TOP10 的某一个进行查询，子查询有路径依赖关系，用户查掉上一级查询则子查询同时失效

过滤条件补充

  • 过滤条件支持「平均加载时间」、「访问量」；

  • 过滤条件默认为「无」；用户选择条件后将条件选择条件展示出来；

  • >,<,= 三个条件;

  • 平均加载时间下拉数值选择范围：0s、1s、2s、……15s

  • 访问量下拉数值选择范围：50，100，200，500，1000，1500，2000，3000，4000，5000
